Project Monster Mansion, later renamed to purgatorium, was my second project working with Lost Annimation. It was created as an entry into the B.P.A. (Business Profesionals of America) hosted by Software Engineering competition in 2020. Unfortunately this project was not showcased at the national competition, due to the Covid-19 outbreak. Purgatiorium was able to be built and played on both Windows and Mac devices using unity's built in compiler. It was written in C# as that was the only programming language officialy supported by Unity.
